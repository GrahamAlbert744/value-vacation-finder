\# Value Vacation Finder



Value Vacation Finder is a Python data science project that identifies potentially undervalued vacation opportunities from Boston.



The project is inspired by value-investing and value-property screening workflows. Instead of screening stocks or real estate, it screens possible vacations by comparing the actual estimated cost of a trip against comparable benchmark trips.



\## Core Question



Can we find safe, attractive, high-quality vacations where the combined cost of flights, lodging, and activities is meaningfully below comparable alternatives?



\## Initial Scope



\- Origin city: Boston

\- Preferred airport: BOS

\- Travelers: 2 adults

\- Trip length: 7 to 21 days

\- Update frequency: manual only

\- MVP mode: destination watchlist search



\## Primary Sources



\- Skyscanner: flights

\- Expedia: hotels and flight/hotel comparison data

\- Tripadvisor: hotels, reviews, and lodging quality

\- Viator: tours, attractions, and experiences

\- Travel Advisory: safety and risk screening



\## Core Outputs



\- Ranked vacation candidates

\- Total estimated trip cost

\- Estimated fair value

\- Estimated discount percentage

\- Travel advisory flags

\- Final recommendation tier
