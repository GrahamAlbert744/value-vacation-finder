\# Project Prompts



This file stores reusable ChatGPT prompts for each phase of the Value Vacation Finder project.



\## Phase 0 Prompt: Setup



You are helping me set up a Python data science project called value-vacation-finder using Cookiecutter Data Science principles.



I am working in Anaconda Prompt on Windows. I want to create a GitHub repo, a conda environment, a clean folder structure, and an initial README.



Guide me one step at a time. For each step:



1\. Tell me exactly what command to run.

2\. Tell me what result I should expect.

3\. Tell me what to do if there is an error.

4\. Tell me when to commit to GitHub.



Do not move ahead until I confirm each step is complete.
